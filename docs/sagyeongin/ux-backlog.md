# UX 백로그 (안정화 후 일괄 처리)

> MVP 동작은 됨. 아래는 "되긴 하나 다듬을 것" — 실사용 중 발견, 도구 안정화 후 일괄.

## 1. 내부 파이프라인 용어 노출
응답에 `stage2_killer` / `stage3_srim` 등 내부 단계명이 사용자에게 그대로 노출.
→ 사용자 말로 치환 ("재무 건전성 검사" / "적정가 계산" 등). 발견: 2026-06 스캔 견적.

## 2. preview 견적 ≠ execute 실제 universe
scan_preview의 `estimated_universe`는 **name-filter 후** 값(예: 286). execute는 정적 필터(상장·거래가능 등) 추가 적용으로 더 줄어듦(예: 216). 견적과 실제가 차이.
→ 버그 아님(과대추정은 안전 방향). 단 사용자 신뢰 위해 preview가 정적 필터도 반영하거나, "견적은 상한" 명시 고려. 발견: 2026-06-19 기계/반도체 스캔(견적 286 vs 실제 216).

## 3. ROE/K 단위 혼용
avg_roe는 %(12.5), K는 분수(0.0742). 내부는 그대로 두되 사용자 표시 시 단위 통일("ROE 12.5% / K 7.4%"). 발견: ADR ROE 노출(03bf087).
